# hubot-rpg

A passive rpg for your chat

See [`src/hubot-rpg.coffee`](src/hubot-rpg.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-rpg --save`

Then add **hubot-rpg** to your `external-scripts.json`:

```json
[
  "hubot-rpg"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
